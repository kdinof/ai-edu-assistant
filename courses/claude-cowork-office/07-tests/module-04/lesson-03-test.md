## Вопросы

### Вопрос 1

**Тип:** scenario

Каждый понедельник у вас еженедельная планёрка. Вы тратите 45 минут на подготовку: открываете таблицу задач, считаете статусы, пишете повестку, формулируете тезисы, создаёте шаблон протокола. Как Cowork может помочь?

- A) Cowork сделает каждый шаг подготовки отдельно — по одному заданию на каждый
- B) Описать весь пайплайн «до встречи» одним заданием: Cowork возьмёт данные из Sheets, создаст повестку с тезисами и шаблон протокола в одном Google Doc
- C) Cowork может только создать повестку — данные из таблиц придётся собирать вручную
- D) Лучше отказаться от повестки и подготовиться устно — так быстрее

**Правильный ответ:** B
**Пояснение:** Подготовка к встрече — идеальный пайплайн для Cowork. Одно задание: данные из Google Sheets (статус задач) → повестка с тезисами → шаблон протокола. Один документ в Google Docs, готовый за 10 минут вместо 45.

### Вопрос 2

**Тип:** find_error

После планёрки коллега записал заметки на бумаге. Потом потратил 20 минут, вручную создавая протокол в Google Docs: набирал текст, форматировал таблицу задач, вспоминал сроки. Что можно было сделать эффективнее?

- A) Записывать заметки более подробно, чтобы потом было проще создать протокол
- B) Не создавать протокол — достаточно заметок
- C) Продиктовать заметки Cowork (даже в свободной форме) и попросить создать структурированный протокол в Google Docs с таблицей action items
- D) Использовать готовый шаблон протокола и заполнять его во время встречи

**Правильный ответ:** C
**Пояснение:** Cowork превращает сырые заметки в структурированный протокол: принятые решения (список), таблица action items (задача | ответственный | срок | статус). Даже хаотичные заметки — достаточный вход для создания полноценного документа.

### Вопрос 3

**Тип:** single_choice

Сколько времени экономит Cowork на подготовке к еженедельной планёрке по сравнению с ручным подходом?

- A) 5-10 минут — незначительная разница
- B) 35-50 минут — вместо 45-60 минут ручной работы нужно 10 минут с Cowork
- C) 0 минут — время одинаковое, просто распределено иначе
- D) Зависит от количества слайдов в презентации

**Правильный ответ:** B
**Пояснение:** Ручная подготовка (сбор данных, повестка, тезисы, шаблон протокола) занимает 45-60 минут. С Cowork: написать задание (3 мин) + проверить план (2 мин) + проверить результат (5 мин) = 10 минут. Экономия — 35-50 минут каждую неделю.

### Вопрос 4

**Тип:** scenario

У вас есть задание для подготовки к планёрке, которое вы используете каждую неделю — только меняете даты. Как лучше всего организовать его для регулярного использования?

- A) Каждый раз копировать задание из прошлой сессии Cowork
- B) Записать задание в заметки на телефоне и копировать оттуда
- C) Сохранить как skill — готовую инструкцию, которую можно запускать одной командой
- D) Попросить Cowork запомнить задание навсегда

**Правильный ответ:** C
**Пояснение:** Skill — это сохранённая инструкция для регулярной задачи. Вместо того чтобы каждый раз искать и копировать задание, вы запускаете skill одной командой. В модуле 5 вы научитесь создавать skills для повторяющихся задач.
