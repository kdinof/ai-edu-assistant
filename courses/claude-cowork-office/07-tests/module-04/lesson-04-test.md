## Вопросы

### Вопрос 1

**Тип:** single_choice

Какой минимальный набор артефактов должен создать полный пайплайн?

- A) 1 артефакт — презентация в Google Slides
- B) 2 артефакта — таблица в Sheets и документ в Docs
- C) 3 связанных артефакта — лист анализа в Sheets, отчёт в Docs, презентация в Slides
- D) 5 артефактов — по одному для каждого инструмента Google Workspace

**Правильный ответ:** C
**Пояснение:** Полный пайплайн включает минимум 3 связанных артефакта из одного задания: аналитический лист в Google Sheets (данные), отчёт в Google Docs (выводы) и презентация в Google Slides (тезисы для показа). Все они связаны общим источником данных.

### Вопрос 2

**Тип:** scenario

Вы создали пайплайн: данные из Sheets → отчёт в Docs → презентация в Slides. Презентацию вы планируете показать клиенту. Перед показом вы замечаете, что в слайде «Ключевые метрики» видна маржинальность по каждому продукту. Что делать?

- A) Показать как есть — клиент оценит прозрачность
- B) Убрать слайд с метриками целиком, чтобы не рисковать
- C) Удалить конфиденциальные данные (маржинальность) из презентации и использовать только агрегированные показатели (общая выручка, количество сделок)
- D) Создать новый пайплайн с нуля, указав в задании не включать маржинальность

**Правильный ответ:** C
**Пояснение:** Маржинальность — внутренние данные, которые не должны попадать в клиентские презентации. Достаточно удалить конфиденциальные метрики и оставить агрегированные показатели. Пересоздавать пайплайн не нужно — быстрее отредактировать готовую презентацию.

### Вопрос 3

**Тип:** find_error

Коллега выполнил задание: создал отчёт в Google Docs и презентацию в Google Slides. Но в отчёте выручка за март — 45 000 000 сум, а в презентации на слайде — 42 000 000 сум. Почему могла возникнуть ошибка?

- A) Cowork иногда генерирует случайные числа — нужно всегда перепроверять все цифры
- B) Скорее всего, пайплайн был описан не одним заданием, а по шагам — и на разных шагах Cowork мог использовать разные источники данных
- C) Google Slides автоматически округляет числа при вставке
- D) Данные в Google Sheets изменились между созданием отчёта и презентации

**Правильный ответ:** B
**Пояснение:** Когда пайплайн описан одним заданием, Cowork берёт данные из одного источника — цифры совпадают во всех артефактах. При пошаговом подходе Cowork может интерпретировать данные по-разному на каждом шаге. Поэтому важно описывать весь пайплайн целиком.
