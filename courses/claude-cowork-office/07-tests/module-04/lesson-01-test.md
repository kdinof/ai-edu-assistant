## Вопросы

### Вопрос 1

**Тип:** single_choice

Вам нужно подготовить презентацию с квартальным отчётом для еженедельной встречи с руководителем. Какой подход к созданию презентации оптимален?

- A) PPTX через code execution — для максимального контроля над дизайном
- B) Google Slides через Google Workspace CLI — быстро, сразу доступна в браузере
- C) Создать слайды вручную в Google Slides — Cowork не умеет работать с презентациями
- D) Создать PDF с тезисами вместо презентации

**Правильный ответ:** B
**Пояснение:** Для рабочих презентаций (отчёт руководителю, повестка встречи) оптимален Google Slides через gws: файл сразу в Google Drive, можно совместно редактировать, не нужно ничего скачивать. PPTX через code execution — для случаев, когда важен сложный дизайн (презентация клиенту, конференция).

### Вопрос 2

**Тип:** scenario

Вы — менеджер проектов в компании Artel. Нужно создать презентацию для партнёра по итогам совместного проекта. Данные в Google Sheets, выводы в Google Docs. Какое задание для Cowork будет наиболее полным?

- A) «Сделай презентацию по проекту»
- B) «Создай презентацию в Google Slides на 8 слайдов. Возьми данные из Sheets "Проект_Artel_Q1" и выводы из Docs "Отчёт_Q1". Структура: титульный, цели проекта, ключевые метрики, динамика, результаты, проблемы, план, вопросы. Speaker notes к каждому слайду. Сохрани в папку "Презентации"»
- C) «Создай красивую презентацию с анимациями на основе данных о проекте»
- D) «Скопируй данные из таблицы в Google Slides, по одной таблице на слайд»

**Правильный ответ:** B
**Пояснение:** Вариант B содержит все элементы фреймворка: источники данных (Sheets + Docs), формат (8 слайдов с конкретной структурой), дополнительные требования (speaker notes) и место сохранения. Вариант C просит анимации, которые Cowork не поддерживает.

### Вопрос 3

**Тип:** multiple_choice

Что Claude Cowork делает хорошо при создании презентаций? (Выберите 3 правильных ответа)

- A) Логичная структура слайдов с ключевыми тезисами
- B) Сложный визуальный дизайн с градиентами и анимациями
- C) Таблицы с данными из Google Sheets
- D) Speaker notes для выступления

**Правильный ответ:** A, C, D
**Пояснение:** Claude отлично справляется с контентом презентации: структурирует слайды, создаёт краткие тезисы, вставляет таблицы с данными и пишет speaker notes. Сложный дизайн, анимации и нестандартные макеты — ограничения, которые лучше доработать вручную.

### Вопрос 4

**Тип:** find_error

Коллега создал презентацию через Cowork для встречи с клиентом. Показывает вам: «Смотри, всё готово — 10 слайдов с данными и выводами. Завтра покажу клиенту прямо так». Что стоит проверить?

- A) Ничего — Cowork создал презентацию, значит всё корректно
- B) Визуальное оформление: Cowork создаёт контент, но базовый дизайн может не подходить для клиентской встречи — стоит доработать вручную или применить корпоративный шаблон
- C) Нужно пересоздать презентацию через PPTX, потому что Google Slides некачественный
- D) Нужно убрать speaker notes — клиент не должен их видеть

**Правильный ответ:** B
**Пояснение:** Claude создаёт контент презентации (структура, тезисы, таблицы), но дизайн будет базовым. Для клиентской встречи стоит применить корпоративный шаблон или доработать оформление вручную. Speaker notes (D) не отображаются при показе — это подсказки для выступающего.
