## Вопросы

### Вопрос 1

**Тип:** scenario

Вы готовитесь создать отчёт через Cowork на основе реальных рабочих данных. В вашей таблице Google Sheets есть колонки: ФИО клиента, телефон, сумма заказа, дата. Что нужно сделать перед тем, как дать задание Cowork?

- A) Ничего — Cowork работает в защищённой среде, данные в безопасности
- B) Удалить таблицу и создать новую только с агрегированными данными
- C) Анонимизировать персональные данные: заменить ФИО на «Клиент 1, 2, 3», убрать телефоны
- D) Скопировать данные в текстовый файл, чтобы Cowork не имел доступа к оригиналу

**Правильный ответ:** C
**Пояснение:** Перед загрузкой данных в AI-инструмент нужно проверить наличие персональных данных (ФИО + контакты). Анонимизация — замена имён на «Клиент 1», удаление телефонов — сохраняет полезность данных для анализа, но защищает конфиденциальность.

### Вопрос 2

**Тип:** single_choice

Какой критерий подтверждает, что вы правильно используете Cowork для создания документов, а не как обычный чат-бот?

- A) Cowork отвечает быстрее, чем ChatGPT
- B) Документы создаются прямо в Google Docs через Google Workspace CLI, без copy-paste
- C) Каждое задание содержит не менее 5 предложений
- D) Cowork предлагает альтернативные варианты документа на выбор

**Правильный ответ:** B
**Пояснение:** Ключевое отличие агентного подхода — результат появляется непосредственно в рабочих инструментах (Google Docs, Sheets) через CLI, без ручного копирования. Если вы копируете текст из чата в документ — вы используете Cowork как чат-бот.

### Вопрос 3

**Тип:** multiple_choice

Какие элементы должно содержать каждое задание для Cowork по фреймворку из этого модуля? (Выберите 4 правильных ответа)

- A) Что сделать — конкретное действие
- B) Откуда взять данные — источник информации
- C) Сколько времени потратить — ограничение по минутам
- D) Куда положить результат — место сохранения
- E) В каком формате — структура документа

**Правильный ответ:** A, B, D, E
**Пояснение:** Фреймворк постановки задачи: Что (действие) → Откуда (источник данных) → Куда (место сохранения) → Формат (структура результата). Ограничение по времени не входит во фреймворк — Cowork сам определяет, сколько времени нужно на выполнение.
