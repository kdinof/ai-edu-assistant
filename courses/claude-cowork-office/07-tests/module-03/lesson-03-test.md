## Вопросы

### Вопрос 1

**Тип:** find_error

Коллега попросил Cowork: «Создай таблицу для бюджета проекта». Получил таблицу с двумя колонками — «Статья» и «Сумма» — без формул и итогов. Коллега недоволен. Что он сделал не так?

- A) Cowork не умеет создавать таблицы с формулами — это нужно делать вручную
- B) Не указал структуру: какие колонки (план/факт/отклонение), какие формулы (суммы, проценты), какие категории расходов
- C) Нужно было сначала создать пустую таблицу, а потом отдельным заданием добавить формулы
- D) Слово «бюджет» слишком общее — нужно было написать «финансовый план»

**Правильный ответ:** B
**Пояснение:** «Таблица для бюджета» — абстрактное задание. Cowork не может угадать, что нужны колонки «План», «Факт», «Отклонение», итоговая строка с суммами и формулы расчёта отклонений. Чем детальнее описание структуры и формул, тем точнее результат.

### Вопрос 2

**Тип:** scenario

Вы хотите, чтобы Cowork создал в Google Sheets таблицу с формулой: «сумма всех расходов, где категория — Маркетинг и месяц — март». Как лучше сформулировать это в задании?

- A) «Добавь формулу SUMIFS(D:D, B:B, "Маркетинг", C:C, "март") в ячейку E2»
- B) «Посчитай сумму расходов, где категория "Маркетинг" и месяц "март"»
- C) «Добавь формулу для подсчёта маркетинговых расходов»
- D) «Нужна сумма за март»

**Правильный ответ:** B
**Пояснение:** Описывайте формулы словами, а не синтаксисом. Cowork сам выберет подходящую функцию (SUMIFS) и правильно укажет диапазоны и условия. Вариант A требует знания синтаксиса и номеров колонок, C и D слишком расплывчаты.

### Вопрос 3

**Тип:** single_choice

Вы создали через Cowork бюджет проекта в Google Sheets. Теперь нужно использовать эти данные для отчёта руководителю. Что может сделать Cowork?

- A) Ничего — данные из Sheets нельзя использовать в других инструментах через MCP
- B) Взять данные из созданной таблицы Sheets и сформировать отчёт в Google Docs с ключевыми цифрами и выводами
- C) Только скопировать таблицу целиком в документ Google Docs
- D) Только отправить таблицу по email через Gmail

**Правильный ответ:** B
**Пояснение:** Cowork через MCP работает и с Sheets, и с Docs. Он может прочитать данные из таблицы и создать документ-отчёт с ключевыми метриками, выводами и рекомендациями. Это связка «таблица → документ» — основа пайплайнов из модуля 4.

### Вопрос 4

**Тип:** scenario

Вам нужен трекер задач для отдела из 8 человек. Какое задание для Cowork создаст наиболее полезную таблицу?

- A) «Создай трекер задач»
- B) «Создай Google Sheet "Трекер задач отдела маркетинга". Колонки: Задача, Ответственный, Дедлайн, Статус (Новая/В работе/На проверке/Готово), Приоритет (Высокий/Средний/Низкий), Комментарий. Добавь лист "Сводка" с подсчётом задач по каждому статусу и по каждому ответственному»
- C) «Создай таблицу с колонками: задача, кто делает, когда, статус»
- D) «Создай трекер задач как в Trello, но в Google Sheets»

**Правильный ответ:** B
**Пояснение:** Вариант B описывает конкретную структуру (6 колонок с конкретными значениями для выпадающих списков) и дополнительный лист «Сводка» с формулами подсчёта. Cowork создаст готовый инструмент, который можно использовать сразу — без доработок.
