### Вопрос 1

**Тип:** scenario

Вы готовитесь проанализировать таблицу расходов компании через Cowork. В таблице есть колонки: контрагент (название компании-поставщика), сумма, дата, категория. Зарплаты сотрудников — на отдельном листе с ФИО и суммами. Что нужно сделать перед анализом?

- A) Удалить лист с зарплатами из таблицы, чтобы Cowork не видел эти данные
- B) Ничего — Cowork работает в защищённой среде, все данные в безопасности
- C) Обезличить зарплатный лист (убрать ФИО, оставить должности), а таблицу расходов с контрагентами можно использовать как есть
- D) Создать копию таблицы и удалить из неё все суммы

**Правильный ответ:** C
**Пояснение:** Агрегированные данные (расходы по категориям, суммы по контрагентам) безопасны для анализа. Но зарплаты с привязкой к ФИО — конфиденциальная информация. Обезличивание (замена ФИО на должности или «Сотрудник 1, 2, 3») сохраняет аналитическую ценность данных.

### Вопрос 2

**Тип:** single_choice

Какой из трёх шагов практического задания формирует первый мини-пайплайн (связку между инструментами)?

- A) Шаг 1: анализ существующей таблицы в Google Sheets
- B) Шаг 2: создание сводной таблицы с формулами
- C) Шаг 3: создание документа в Google Docs на основе данных из Google Sheets
- D) Все три шага вместе формируют пайплайн

**Правильный ответ:** C
**Пояснение:** Шаг 3 — это первый пайплайн: данные из Sheets → документ в Docs. Cowork берёт результаты анализа из таблицы и формирует отчёт в Google Docs. Шаги 1 и 2 работают внутри одного инструмента (Sheets), а Шаг 3 связывает два инструмента через MCP.

### Вопрос 3

**Тип:** find_error

Коллега выполнил практическое задание: проанализировал таблицу, Cowork выдал результат прямо в чате — текстом с цифрами. Коллега скопировал ответ в Google Docs и говорит: «Готово, пайплайн работает!» Что не так?

- A) Результат анализа должен быть на новом листе в Google Sheets, а не текстом в чате — Cowork должен работать через MCP, без copy-paste
- B) Коллега не добавил формулы в Google Docs — документ должен быть интерактивным
- C) Нужно было использовать два разных чата Cowork — один для Sheets, другой для Docs
- D) Всё правильно — главное, что результат есть в Google Docs

**Правильный ответ:** A
**Пояснение:** Весь смысл работы через MCP — без copy-paste. Результат анализа должен быть на отдельном листе в Google Sheets (через MCP), а документ в Docs — тоже создан Cowork напрямую. Если вы копируете текст из чата, вы используете Cowork как чат-бот, а не как агент.
