## Вопросы

### Вопрос 1

**Тип:** multiple_choice

Какие компоненты должен содержать готовый AI-офис по итогам курса? (Выберите 4 правильных ответа)

- A) Claude Cowork с подключённым Google Workspace CLI
- B) CLAUDE.md с контекстом должности и правилами
- C) Аккаунт ChatGPT Plus как резервный инструмент
- D) Библиотека skills (минимум 3) для регулярных задач
- E) Time Audit с замерами «до» и «после»

**Правильный ответ:** A, B, D, E
**Пояснение:** Готовый AI-офис включает: (1) настроенный Cowork с Google Workspace CLI, (2) CLAUDE.md с контекстом, (3) минимум 3 skills для повторяющихся задач, (4) Time Audit с измеренной экономией. ChatGPT не является частью этой системы.

### Вопрос 2

**Тип:** scenario

Через месяц после курса вы перешли в другой отдел. Задачи изменились: вместо отчётов по продажам — управление проектами. Какие компоненты AI-офиса нужно обновить?

- A) Только CLAUDE.md — остальное работает универсально
- B) CLAUDE.md (новая должность и задачи) и skills (новые задачи → новые шаблоны). Cowork и Google Workspace CLI остаются как есть
- C) Нужно проходить курс заново с нуля
- D) Ничего — AI-офис автоматически адаптируется к новым задачам

**Правильный ответ:** B
**Пояснение:** При смене должности обновите CLAUDE.md (новый контекст, задачи, формат документов) и создайте новые skills для повторяющихся задач на новой позиции. Cowork и Google Workspace CLI — инфраструктура, которая остаётся прежней. AI-офис растёт вместе с вами.

### Вопрос 3

**Тип:** single_choice

Вы заметили новую повторяющуюся задачу: каждую среду нужно сводить данные из трёх разных Google Sheets в один отчёт. Что лучше сделать?

- A) Каждую среду формулировать задание заново — так точнее
- B) Сначала выполнить задачу вручную через Cowork, убедиться в результате, затем сохранить как skill
- C) Сразу создать сложный skill с 10 шагами, чтобы покрыть все возможные сценарии
- D) Попросить коллегу из IT-отдела написать скрипт для автоматизации

**Правильный ответ:** B
**Пояснение:** Правило создания skills: сначала проверьте задание на практике, убедитесь в хорошем результате, потом сохраните как skill. Это гарантирует, что skill работает правильно с первого запуска. AI-офис растёт по мере появления новых повторяющихся задач.
