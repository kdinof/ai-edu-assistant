## Паспорт курса

| Параметр | Значение |
|----------|----------|
| Название | Claude Cowork для офисных работников |
| Slug | claude-cowork-office |
| Длительность | ~2.5 часа (5 модулей x ~30 мин) |
| Формат | Видео-скринкасты (5-15 мин/урок) + текстовый дубль |
| Уровень | Начинающий |
| Основной инструмент | Claude Cowork (Desktop, Mac) |
| Вспомогательный | Google Workspace MCP Server |

| Финальный артефакт | Работающий персональный AI-офис: Claude Cowork + Google Workspace MCP + библиотека skills + автоматизированные задачи + Time Audit с измеримой экономией |

## Ключевая идея курса

**Claude Cowork — не чат-бот, а агент.** Разница принципиальна:
- Чат-бот (ChatGPT, базовый Claude): ты пишешь промпт → получаешь текст → копируешь в документ
- Агент (Cowork): ты ставишь задачу → Cowork составляет план → ты одобряешь → Cowork выполняет работу прямо в твоих файлах и Google Workspace

Этот курс учит второму подходу. Студент не учится "общаться с AI" — он настраивает AI-ассистента, который работает с его реальными инструментами.

## Сквозной проект

Студент пошагово строит **персональный AI-офис** — систему, где Claude Cowork подключён к рабочим инструментам и выполняет задачи автономно:

| Модуль | Вклад в сквозной проект |
|--------|------------------------|
| 1 | Настроенная среда: Cowork + Google Workspace MCP + CLAUDE.md с контекстом должности + Time Audit "до" |
| 2 | Работа с документами: Claude читает и создаёт Google Docs напрямую. Отчёты, письма, протоколы |
| 3 | Работа с данными: Claude анализирует Google Sheets, создаёт таблицы с формулами, строит визуализации |
| 4 | Презентации и пайплайны: данные из Sheets → анализ → отчёт в Docs → презентация. Полный цикл через MCP |
| 5 | Автоматизация: повторяющиеся задачи, финальная библиотека skills, Time Audit "после" |

## Цели обучения (Learning Outcomes)

После прохождения курса студент сможет:

1. **Настроить** персональную рабочую среду: Claude Cowork + Google Workspace MCP + CLAUDE.md с контекстом должности и правилами безопасности данных
2. **Подключить** Claude к Google Workspace через MCP — чтобы Claude читал, создавал и редактировал Google Docs, Sheets и Slides напрямую, без копирования
3. **Применять** агентный подход к задачам: декомпозировать рабочую задачу → поставить её Cowork → одобрить план → получить готовый артефакт в Google Workspace
4. **Собирать** пайплайны из нескольких шагов: данные (Sheets) → анализ → отчёт (Docs) → презентация (Slides), где Claude выполняет всю цепочку
5. **Оценить** экономию времени через Time Audit и определять, для каких задач AI подходит (task-fit), а для каких нет

## Целевая аудитория

### Основной сегмент
- **Кто:** Офисные работники в Узбекистане — бухгалтеры, менеджеры проектов, маркетологи, финансисты, HR-специалисты. 25-45 лет.
- **Боль:** Много времени уходит на рутину — отчёты, переписка, подготовка презентаций, сбор данных. Не хватает времени на стратегические задачи.
- **Мотивация:** Ускорить рутину, высвободить время для работы, где нужна экспертиза. Microsoft фиксирует ~29% рост продуктивности у пользователей AI.
- **Контекст:** Учится в свободное время или по инициативе работодателя. Формат микро-уроков позволяет проходить в обеденный перерыв или после работы.

### Психологический профиль (из research)
- **Главный барьер — индифферентность, не страх.** Работники не видят конкретной пользы AI для задач, на которых их оценивают. Курс преодолевает это через конкретные use cases с первого модуля.
- **Безопасность данных — #1 тревога.** 70% бухгалтеров обеспокоены безопасностью. Sandboxed-подход Cowork и MCP с permissions — преимущество, встроенное в каждый модуль.
- **Skills gap = problem framing.** Проблема не в промптинге, а в декомпозиции рабочей задачи для AI. Курс учит именно этому: задача → план → выполнение.
- **Работники опережают компании.** ~50% начали использовать AI на работе самостоятельно. Наша ЦА — проактивные самоучки, которым не хватает системного подхода.

### Дополнительный сегмент
- **Кто:** Фрилансеры и предприниматели, работающие с документами и клиентской коммуникацией.

## Scope

### Включено
- Claude Cowork: настройка среды, CLAUDE.md, агентный режим (plan → execute)
- MCP: подключение Google Workspace (Docs, Sheets, Slides, Drive) к Claude
- Агентный подход: декомпозиция задач, одобрение планов, автономное выполнение
- Работа с документами через Google Docs MCP: отчёты, письма, протоколы, анализ PDF
- Работа с таблицами через Google Sheets MCP: создание таблиц с формулами, анализ данных
- Создание презентаций через Google Slides MCP и PPTX
- Пайплайны: данные (Sheets) → анализ → отчёт (Docs) → презентация (Slides) — полный цикл через MCP
- Skills: создание и использование навыков для повторяющихся задач
- Безопасность данных: что можно загружать, MCP permissions, sandboxed-подход (встроено в каждый модуль)
- Task-fit rubric: когда AI подходит, когда нет
- Time Audit: методология замера экономии времени (до/после)
- Практика на реальных рабочих данных студента в каждом модуле

### НЕ включено
- Программирование и написание кода (курс для не-программистов)
- Генерация изображений и дизайн (→ направляем в AI Bundle)
- Claude API и кастомные MCP-серверы (используем только готовый Google Workspace MCP)
- Microsoft Copilot и ChatGPT (фокус на Claude — глубина, не ширина)
- Работа с конфиденциальными/регулируемыми данными (compliance)
- Claude Code (терминальный режим для разработчиков)

## Топики и модули (высокоуровнево)

| # | Модуль | Фокус | Опирается на | Вклад в сквозной проект | Время |
|---|--------|-------|-------------|------------------------|-------|
| 1 | Фундамент: настройка AI-офиса | Установка Cowork, подключение Google Workspace MCP, CLAUDE.md, task-fit rubric, Time Audit "до" | — | Настроенная среда с MCP + контекст должности | ~30 мин |
| 2 | Документы и отчёты | Создание и анализ документов через Google Docs MCP. Отчёты, письма, протоколы, анализ PDF | Модуль 1 (среда + MCP) | Skills для документов, работа с реальными docs | ~30 мин |
| 3 | Таблицы и данные | Анализ и создание таблиц через Google Sheets MCP. Формулы, сводные таблицы, визуализация | Модуль 2 (навык агентной работы) | Skills для таблиц, интеграция с документами | ~30 мин |
| 4 | Презентации и пайплайны | Создание презентаций. Полный пайплайн: Sheets → анализ → Docs отчёт → Slides презентация | Модули 2+3 (документы + данные + MCP) | Полный пайплайн через MCP | ~30 мин |
| 5 | Автоматизация и результат | Skills для повторяющихся задач, шаблоны workflow, финальная библиотека, Time Audit "после" | Модули 1-4 (все skills) | Автоматизированная система + измерение результата | ~30 мин |

**Нарастающая сложность:** Модуль 1 → настройка и первые команды. Модуль 2 → один тип документа. Модуль 3 → данные и анализ. Модуль 4 → пайплайн из нескольких шагов через MCP. Модуль 5 → автоматизация и масштабирование. Каждый модуль невозможно пройти без предыдущего.

**Роль MCP в каждом модуле:**
- Модуль 1: подключение и настройка Google Workspace MCP
- Модуль 2: Claude читает/создаёт Google Docs через MCP
- Модуль 3: Claude работает с Google Sheets через MCP
- Модуль 4: Claude оркестрирует Sheets → Docs → Slides через MCP
- Модуль 5: MCP как основа автоматизированных workflow

### Ключевой вывод

**Пустая ниша:** нет ни одного курса, который учит Claude Cowork + MCP для офисных задач. Все конкуренты используют чат-ботный подход (промпт → текст → копируй). Наш курс учит агентный подход (задача → план → выполнение прямо в рабочих инструментах).

### Дифференциация OSNOVA

1. **Агентный подход вместо чат-бота** — Cowork выполняет задачи прямо в Google Workspace через MCP, а не генерирует текст для копирования
2. **Единственный курс по Cowork + MCP для офиса** — конкуренты учат ChatGPT; Claude Cowork-курсов для нетехнических специалистов нет
3. **Сквозной проект** — студент строит работающий AI-офис, где каждый модуль добавляет новый capability
4. **Практика на своих данных** — студент подключает свой Google Workspace с первого модуля (research: #1 best practice)
5. **Безопасность как преимущество** — sandboxed Cowork + MCP permissions (адресует #1 тревогу ЦА — 70% обеспокоены безопасностью)
6. **Измеримый результат** — Time Audit до/после с конкретными задачами и минутами
7. **First-mover в Узбекистане** — нет AI productivity курсов для non-tech специалистов

## Риски и допущения

- **Риск:** Claude Cowork — research preview, интерфейс может измениться → **Митигация:** Модульная структура + текстовые дубли позволяют обновить отдельные уроки без пересъёмки. Ядро (агентный подход, MCP, task-fit) стабильно
- **Риск:** Cowork доступен только на Mac → **Митигация:** Указать в пререквизитах. Для Windows-пользователей: базовые задачи модулей 2-3 выполнимы через claude.ai + Projects, но без агентного режима
- **Риск:** Google Workspace MCP Server может быть нестабилен или измениться → **Митигация:** MCP — открытый стандарт Anthropic, Google официально поддерживает. Подготовить fallback-инструкции для работы через файловую систему
- **Риск:** Платная подписка Claude ($20-100/мес) — барьер для ЦА в Узбекистане → **Митигация:** В модуле 1 конкретный расчёт ROI через Time Audit. Research: AI даёт ~29% рост продуктивности
- **Риск:** Индифферентность ЦА — "не вижу как это поможет мне" (главный барьер по research) → **Митигация:** Каждый модуль начинается с конкретной узнаваемой задачи, а не с теории
- **Риск:** Студенты переоценят AI ("jagged frontier") → **Митигация:** Task-fit rubric в модуле 1. Research: "AI интенсифицирует работу, а не снижает" (HBR) — быть честными
- **Риск:** Безопасность данных при подключении Google Workspace → **Митигация:** MCP permissions, правила в каждом модуле (какие данные можно, какие нет), sandboxed-подход Cowork
- **Риск:** Google Workspace Studio (Gemini 3) как встроенный конкурент → **Митигация:** Cowork даёт больше контроля, глубины и кастомизации, чем встроенный Gemini. Позиционировать как "персональный AI-ассистент" vs "функция в продукте"
- **Допущение:** ЦА имеет Mac для установки Claude Desktop
- **Допущение:** ЦА имеет Google Workspace аккаунт (бесплатный достаточен)
- **Допущение:** Приглашённый эксперт доступен для записи скринкастов
