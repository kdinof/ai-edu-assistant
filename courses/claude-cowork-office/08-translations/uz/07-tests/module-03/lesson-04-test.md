# Test: Jadvallar va ma'lumotlar / Dars 3.4: Amaliyot: o'z ma'lumotlarimizni tahlil qilamiz

## Savollar

### Savol 1

**Turi:** scenario

Siz kompaniya xarajatlari jadvalini Cowork orqali tahlil qilishga tayyorlanmoqdasiz. Jadvalda ustunlar bor: kontragent (yetkazib beruvchi kompaniya nomi), summa, sana, turkum. Xodimlar ish haqi — alohida varaqda, FIO va summalar bilan. Tahlildan oldin nima qilish kerak?

- A) Ish haqi varag'ini jadvaldan o'chirish, Cowork bu ma'lumotlarni ko'rmasligi uchun
- B) Hech narsa — Cowork himoyalangan muhitda ishlaydi, barcha ma'lumotlar xavfsiz
- C) Ish haqi varag'ini shaxsiylashtirilmagan holga keltirish (FIO ni olib tashlab, lavozimlarni qoldirish), kontragentli xarajatlar jadvalini esa bor holaticha ishlatish mumkin
- D) Jadval nusxasini yaratib, undan barcha summalarni o'chirish

**To'g'ri javob:** C
**Izoh:** Yig'ma ma'lumotlar (turkumlar bo'yicha xarajatlar, kontragentlar bo'yicha summalar) tahlil uchun xavfsiz. Lekin FIO ga bog'langan ish haqi — maxfiy ma'lumot. Shaxsiylashtirilmagan holga keltirish (FIO ni lavozimlar yoki «Xodim 1, 2, 3» bilan almashtirish) ma'lumotlarning tahliliy qiymatini saqlaydi.

### Savol 2

**Turi:** single_choice

Amaliy topshiriqning qaysi qadami birinchi mini-payplaynni (vositalar orasidagi bog'lanishni) shakllantiradi?

- A) 1-qadam: Google Sheets dagi mavjud jadvalni tahlil qilish
- B) 2-qadam: formulali xulosa jadvali yaratish
- C) 3-qadam: Google Sheets dagi ma'lumotlar asosida Google Docs da hujjat yaratish
- D) Barcha uchala qadam birgalikda payplaynni shakllantiradi

**To'g'ri javob:** C
**Izoh:** 3-qadam — bu birinchi payplayn: Sheets dagi ma'lumotlar → Docs dagi hujjat. Cowork tahlil natijalarini jadvaldan olib, Google Docs da hisobot shakllantiradi. 1 va 2-qadamlar bitta vosita ichida ishlaydi (Sheets), 3-qadam esa ikkita vositani MCP orqali bog'laydi.

### Savol 3

**Turi:** find_error

Hamkasbingiz amaliy topshiriqni bajardi: jadvalni tahlil qildi, Cowork natijani to'g'ridan-to'g'ri chatda — raqamlar bilan matn shaklida — berdi. Hamkasbingiz javobni Google Docs ga nusxaladi va aytdi: «Tayyor, payplayn ishlayapti!» Nima noto'g'ri?

- A) Tahlil natijasi chatdagi matn emas, Google Sheets dagi yangi varaqda bo'lishi kerak — Cowork MCP orqali, nusxalashsiz ishlashi kerak
- B) Hamkasbingiz Google Docs ga formulalar qo'shmadi — hujjat interaktiv bo'lishi kerak edi
- C) Ikkita alohida Cowork chati ishlatish kerak edi — biri Sheets uchun, biri Docs uchun
- D) Hammasi to'g'ri — asosiysi natija Google Docs da bo'lishi

**To'g'ri javob:** A
**Izoh:** MCP orqali ishlashning butun mohiyati — nusxalashsiz. Tahlil natijasi Google Sheets da alohida varaqda (MCP orqali) bo'lishi kerak, Docs dagi hujjat ham Cowork tomonidan bevosita yaratilishi kerak. Agar siz chatdan matnni nusxalayotgan bo'lsangiz, Cowork ni agent sifatida emas, chat-bot sifatida ishlatayotgan bo'lasiz.
