## Видео-скрипт

### Вступление (30 сек)

У вас настроен Cowork и подключён Google Workspace. Claude уже может работать с вашими файлами. Но пока он не знает, кто вы, чем занимаетесь и какие задачи решаете каждый день. Это как нанять нового ассистента и не рассказать ему о компании. В этом уроке мы это исправим — создадим файл CLAUDE.md.

### Основная часть

**Что такое CLAUDE.md**

CLAUDE.md — это текстовый файл с инструкциями для Claude. Он хранится на вашем компьютере и автоматически загружается каждый раз, когда вы открываете Claude Desktop.

Без CLAUDE.md Claude — универсальный AI, который ничего не знает о вашей работе. С CLAUDE.md — персональный ассистент, который понимает вашу должность, формат документов и правила компании.

**Что включить в CLAUDE.md**

Вот структура, которая подходит для офисных работников:

**1. Кто вы.** Должность, отдел, компания. Это помогает Claude понимать контекст.

```
## Обо мне
Я — бухгалтер в компании «Vertex» (строительная отрасль, Ташкент).
Работаю с финансовой отчётностью, счетами, актами сверки.
```

**2. Типичные задачи.** Что вы делаете каждую неделю. Claude будет знать, какие задачи для вас актуальны.

```
## Мои задачи
- Еженедельный отчёт по расходам для директора
- Подготовка счетов на оплату
- Сверка данных между таблицами
- Ответы на запросы от клиентов и поставщиков
```

**3. Формат документов.** Как принято оформлять документы в вашей компании.

```
## Формат документов
- Отчёты: заголовок, дата, таблица с данными, 3 ключевых вывода
- Письма: обращение на «вы», подпись «С уважением, [имя], [должность]»
- Таблицы: первая строка — заголовки, суммы внизу
```

**4. Правила и ограничения.** Что Claude должен и не должен делать.

```
## Правила
- Всегда используй русский язык
- Не выдумывай данные — работай только с тем, что я предоставил
- Если не хватает информации — спроси, а не додумывай
- Не включай в документы персональные данные клиентов
```

**Демонстрация: бухгалтер vs маркетолог**

Посмотрите, как контекст меняет качество работы Claude.

*Задача:* «Подготовь еженедельный отчёт»

*Без CLAUDE.md:* Claude спросит — какой отчёт? для кого? в каком формате? Или сделает что-то общее и бесполезное.

*CLAUDE.md бухгалтера:* Claude знает, что еженедельный отчёт = расходы по категориям, таблица, 3 вывода, для директора. Результат — готовый документ в нужном формате.

*CLAUDE.md маркетолога:* Claude знает, что еженедельный отчёт = метрики кампаний (CTR, конверсия, бюджет), сравнение с прошлой неделей, рекомендации. Совсем другой документ из того же запроса.

**Антипаттерн: пустой CLAUDE.md**

Если CLAUDE.md пустой или отсутствует, вам придётся каждый раз объяснять Claude контекст заново. Это как каждое утро заново рассказывать ассистенту, в какой компании он работает.

Потратьте 10 минут на CLAUDE.md один раз — и сэкономите эти 10 минут при каждом обращении.

### Заключение (30 сек)

CLAUDE.md — это ваша инвестиция в качество работы Claude. Чем точнее вы опишете свою должность, задачи и правила, тем лучше будут результаты. В следующем уроке мы соберём всё вместе: проверим установку, создадим ваш CLAUDE.md и заполним Time Audit «до».

## Текстовое сопровождение

### Что такое CLAUDE.md

CLAUDE.md — текстовый файл с инструкциями для Claude. Хранится локально на вашем Mac и загружается автоматически при каждом запуске Claude Desktop.

**Без CLAUDE.md** → Claude не знает контекст, каждый раз приходится объяснять заново.
**С CLAUDE.md** → Claude понимает вашу должность, задачи и формат документов.

### Структура CLAUDE.md для офисного работника

```markdown
# Мой AI-ассистент

## Обо мне
[Должность], [отдел], [компания], [город].
[Чем занимаетесь в 1-2 предложениях.]

## Мои типичные задачи
- [Задача 1, которую делаете каждую неделю]
- [Задача 2]
- [Задача 3]
- [Задача 4]

## Формат документов
- Отчёты: [как оформлять]
- Письма: [стиль, обращение, подпись]
- Таблицы: [структура]

## Рабочие инструменты
- Google Docs — для отчётов и писем
- Google Sheets — для таблиц и данных
- [Другие инструменты]

## Правила
- Язык: русский
- Не выдумывай данные — работай только с предоставленным
- Если не хватает информации — спроси
- Не включай персональные данные клиентов в документы
- [Другие правила вашей компании]
```

### Примеры для разных должностей

**Бухгалтер:**
```markdown
## Обо мне
Бухгалтер в ООО «Vertex» (строительная отрасль, Ташкент).
Веду учёт расходов, готовлю отчёты для директора, работаю со счетами.

## Мои типичные задачи
- Еженедельный отчёт по расходам (категории, суммы, динамика)
- Подготовка счетов на оплату поставщикам
- Сверка данных между Google Sheets и 1С
- Ответы на запросы по дебиторской задолженности
```

**Маркетолог:**
```markdown
## Обо мне
Маркетолог в компании «DataSoft» (IT-сектор, Ташкент).
Веду рекламные кампании, готовлю отчёты по метрикам, пишу контент.

## Мои типичные задачи
- Еженедельный отчёт по рекламным кампаниям (CTR, CPA, ROAS)
- Подготовка контент-плана на месяц
- Написание текстов для email-рассылок
- Анализ конкурентов
```

**Менеджер проектов:**
```markdown
## Обо мне
Менеджер проектов в «Green Energy Uz» (возобновляемая энергетика, Ташкент).
Координирую 3 проекта, готовлю отчёты для заказчиков, веду протоколы.

## Мои типичные задачи
- Статус-отчёт по проектам каждый понедельник
- Протоколы совещаний с action items
- Подготовка презентаций для заказчиков
- Трекинг задач и дедлайнов команды
```

### Где хранится CLAUDE.md

Файл создаётся в корневой папке, с которой работает Claude Desktop. Вы можете создать его через Cowork:

> «Создай файл CLAUDE.md в текущей директории со следующим содержимым: ...»

Или создать вручную в любом текстовом редакторе.

## Ключевые выводы

1. **CLAUDE.md** — файл-инструкция, который превращает универсальный AI в персонального ассистента. Описывает вашу должность, задачи, формат документов и правила
2. **Контекст меняет качество.** Одна и та же команда «подготовь отчёт» даёт разные результаты для бухгалтера и маркетолога — если Claude знает контекст из CLAUDE.md
3. **Потратьте 10 минут один раз** — и получите лучшие результаты при каждом обращении. Пустой CLAUDE.md = Claude не знает, кто вы и что вам нужно
