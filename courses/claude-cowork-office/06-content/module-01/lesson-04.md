## Видео-скрипт

### Вступление (30 сек)

В предыдущих уроках мы разобрали теорию: чем агент отличается от чат-бота, как установить Cowork и подключить Google Workspace CLI, зачем нужен CLAUDE.md. Теперь давайте всё это сделаем. В этом уроке вы настроите свой AI-офис и заполните Time Audit «до» — зафиксируете, сколько времени занимают ваши текущие задачи.

### Основная часть

**Задание 1: Проверьте установку (2 мин)**

Откройте Claude Desktop и пройдите по чеклисту:

- [ ] Claude Desktop установлен и запущен
- [ ] Вкладка Cowork доступна
- [ ] Google Workspace CLI подключён (Claude видит ваши файлы в Google Drive)

Если Google Workspace CLI ещё не настроен — вернитесь к уроку 1.2 и выполните шаги. Без него остальные модули курса работать не будут.

Проверьте подключение — напишите в Cowork:
> «Покажи список моих файлов в Google Drive»

Claude должен вернуть список ваших файлов.

**Задание 2: Создайте CLAUDE.md (5 мин)**

Создайте CLAUDE.md для своей должности. Используйте шаблон из урока 1.3:

1. Опишите, кто вы: должность, отдел, компания
2. Перечислите 4-5 типичных задач, которые делаете каждую неделю
3. Укажите формат документов, принятый в вашей компании
4. Добавьте правила: язык, ограничения, что Claude не должен делать

Вы можете попросить Cowork помочь:
> «Помоги мне создать CLAUDE.md. Я работаю [должность] в [компания]. Мои основные задачи: [перечислите]. Задай мне уточняющие вопросы, чтобы составить полный файл.»

Проверьте результат — дайте Claude простую задачу и посмотрите, использует ли он контекст из CLAUDE.md:
> «Подготовь шаблон моего еженедельного отчёта»

Если Claude создаёт отчёт, соответствующий вашей должности и формату — CLAUDE.md работает.

**Задание 3: Time Audit «до» (3 мин)**

Time Audit — способ измерить, сколько времени вы тратите на задачи до и после внедрения AI. Без этого вы не сможете оценить реальную пользу.

Выберите **3-5 повторяющихся рабочих задач** и запишите, сколько времени каждая занимает сейчас:

| Задача | Частота | Время сейчас |
|--------|---------|-------------|
| Пример: Еженедельный отчёт | 1 раз/неделю | 2 часа |
| Пример: Подготовка к совещанию | 2 раза/неделю | 40 мин |
| Пример: Деловое письмо | 5 раз/неделю | 20 мин |
| ... | ... | ... |

Правила выбора задач:
- Задачи, которые вы делаете **регулярно** (минимум раз в неделю)
- Задачи, связанные с **документами, таблицами или текстами** (то, что Claude может делать)
- Будьте честны со временем — записывайте реальные цифры, не приблизительные

Создайте эту таблицу в Google Sheets — мы вернёмся к ней в модуле 5, когда будем замерять экономию.

Вы можете попросить Cowork создать её:
> «Создай Google Sheet "Time Audit" с колонками: Задача, Частота, Время до, Время после, Экономия. Заполни первый столбец моими типичными задачами из CLAUDE.md»

**Безопасность: финальная проверка**

Перед тем как двигаться дальше, убедитесь:
- [ ] Через Google Workspace CLI подключён **рабочий** Google аккаунт (не личный с семейными фото)
- [ ] В папках, доступных Claude, нет документов с персональными данными клиентов
- [ ] Вы понимаете, какие файлы Claude может видеть

### Заключение (30 сек)

Поздравляю — ваш AI-офис настроен. У вас есть Claude Cowork с подключённым Google Workspace, CLAUDE.md с контекстом вашей должности и Time Audit «до» с реальными цифрами. В следующем модуле мы начнём использовать этот офис для создания документов — отчётов, писем и протоколов прямо в Google Docs.

## Текстовое сопровождение

### Чеклист настройки AI-офиса

Выполните все пункты перед переходом к модулю 2:

**Установка:**
- [ ] Claude Desktop установлен на Mac
- [ ] Подписка Pro или Max активна
- [ ] Вкладка Cowork доступна

**Google Workspace CLI:**
- [ ] `gws` установлен (`npm install -g @googleworkspace/cli`)
- [ ] `gws auth setup` выполнен
- [ ] `gws auth login` выполнен (Drive, Docs, Sheets)
- [ ] Claude видит файлы в Google Drive

**CLAUDE.md:**
- [ ] Файл создан
- [ ] Указана должность и компания
- [ ] Перечислены типичные задачи
- [ ] Описан формат документов
- [ ] Добавлены правила и ограничения

**Time Audit:**
- [ ] Выбраны 3-5 повторяющихся задач
- [ ] Записано реальное время на каждую
- [ ] Таблица создана в Google Sheets

**Безопасность:**
- [ ] Подключён рабочий (не личный) Google аккаунт
- [ ] В доступных папках нет документов с персональными данными
- [ ] Создана отдельная рабочая папка для Claude (рекомендуется)

### Шаблон Time Audit

Создайте Google Sheet со следующей структурой:

| Задача | Частота | Время «до» | Время «после» | Экономия | Экономия в неделю |
|--------|---------|-----------|--------------|----------|-------------------|
| | | | _заполним в модуле 5_ | _авто_ | _авто_ |

**Как выбрать задачи для Time Audit:**

Подходят:
- Подготовка отчётов (еженедельных, месячных)
- Написание деловых писем
- Подготовка к совещаниям
- Анализ данных в таблицах
- Создание презентаций
- Обработка входящих документов

Не подходят:
- Задачи, требующие личного общения (переговоры, звонки)
- Творческие задачи с высокой неопределённостью
- Задачи с конфиденциальными данными, которые нельзя загружать в AI

### Что дальше

В модуле 2 вы начнёте использовать настроенный AI-офис для реальных задач:
- Создание отчётов в Google Docs
- Деловые письма
- Протоколы совещаний

Все эти задачи Claude будет выполнять напрямую в Google Docs через Google Workspace CLI — без copy-paste.

## Ключевые выводы

1. **AI-офис = Cowork + Google Workspace CLI + CLAUDE.md + Time Audit.** Все четыре компонента нужны для эффективной работы. Без `gws` — нет доступа к файлам. Без CLAUDE.md — нет контекста. Без Time Audit — нет измеримого результата
2. **Time Audit «до»** фиксирует текущее время на задачи. В модуле 5 вы замерите время «после» и увидите конкретную экономию в минутах и часах
3. **Безопасность — не формальность.** Подключайте рабочий аккаунт, контролируйте доступ к папкам, не загружайте документы с персональными данными клиентов

## Практическое задание

**Задание:** Настроить полный AI-офис: Cowork + Google Workspace CLI + CLAUDE.md + Time Audit «до»

**Критерии выполнения:**
- Claude Desktop установлен, Cowork доступен
- Google Workspace CLI подключён, Claude видит файлы в Google Drive
- CLAUDE.md создан с контекстом вашей должности (минимум: кто вы, задачи, формат, правила)
- Time Audit «до» заполнен в Google Sheets (3-5 задач с реальным временем)
- Проверка безопасности пройдена (рабочий аккаунт, нет персональных данных в доступных папках)
