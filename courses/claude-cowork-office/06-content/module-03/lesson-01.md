## Видео-скрипт

### Вступление (30 сек)

Здравствуйте! В прошлом модуле мы научились создавать документы через Cowork и Google Docs. Но документы — это только половина офисной работы. Вторая половина — таблицы: отчёты, бюджеты, аналитика, трекеры.

В этом уроке разберём, что именно Claude Cowork может делать с Google Sheets через MCP — и где проходит граница его возможностей.

### Основная часть

**Что Cowork делает с Google Sheets через MCP**

Когда вы подключили Google Workspace MCP в модуле 1, Cowork получил доступ не только к Docs, но и к Sheets. Вот что он умеет:

**Читать данные.** Cowork может открыть любую таблицу из вашего Google Drive, прочитать содержимое листов, понять структуру данных — заголовки, типы столбцов, количество строк.

**Анализировать.** Вы спрашиваете: «Какие топ-5 категорий расходов за квартал?» — Cowork читает данные, считает суммы, сортирует и даёт ответ. Не просто текстовый — он может создать новый лист с результатом прямо в той же таблице.

**Создавать таблицы с нуля.** «Создай бюджет проекта на Q2 в Google Sheets» — и Cowork создаёт новую таблицу с колонками, строками, форматированием.

**Писать формулы.** Вместо того чтобы вспоминать синтаксис SUMIF или VLOOKUP, вы описываете задачу словами: «Посчитай сумму продаж по каждому менеджеру». Cowork сам напишет формулу и вставит её в нужную ячейку.

**Добавлять данные.** Cowork может дописывать строки в существующую таблицу — например, добавить новые записи или обновить данные.

**Демонстрация: от сырых данных к выводам за минуту**

Допустим, у вас есть таблица продаж за месяц — 300 строк с датами, менеджерами, суммами и категориями товаров.

Задание Cowork: «Открой таблицу "Продажи март" в Google Sheets. Найди топ-3 менеджера по сумме продаж, посчитай динамику по неделям, создай новый лист "Аналитика" с результатами».

Что происходит:
1. Cowork читает таблицу через MCP
2. Группирует данные по менеджерам, считает суммы
3. Разбивает продажи по неделям
4. Создаёт новый лист с двумя таблицами: рейтинг менеджеров и понедельная динамика
5. Добавляет формулы для итогов

Результат — готовый аналитический лист в вашей таблице. Без copy-paste, без ручного построения формул.

**Ограничения: что Cowork делает не очень хорошо**

Важно знать границы:

- **Графики и диаграммы.** Google Sheets MCP пока не поддерживает создание графиков. Cowork подготовит данные, но визуализацию нужно добавить вручную или попросить Cowork создать файл Excel с графиками через code execution.
- **Сложные макросы и скрипты.** Apps Script — не территория MCP. Если вам нужна автоматизация на уровне скриптов, это отдельная задача.
- **Очень большие таблицы (10 000+ строк).** MCP передаёт данные текстом, и на больших объёмах это может быть медленно. Для таких задач лучше предварительно фильтровать данные.
- **Форматирование ячеек.** Cowork может записать данные и формулы, но тонкое форматирование (цвета, заливка, объединение ячеек) пока ограничено.

**Правило:** используйте Cowork для структуры, данных и формул. Финальное визуальное оформление — в самом Google Sheets.

### Заключение (30 сек)

Итак, Cowork через MCP умеет читать, анализировать, создавать таблицы и писать формулы в Google Sheets. Ограничения — графики, макросы и очень большие массивы данных.

## Текстовое сопровождение

### Что Cowork умеет делать с Google Sheets

Claude Cowork подключён к Google Sheets через MCP. Это даёт ему возможность работать с таблицами напрямую — без copy-paste и ручного переноса данных.

| Возможность | Пример |
|-------------|--------|
| Чтение данных | Открыть таблицу, прочитать содержимое любого листа |
| Анализ | Посчитать суммы, найти тренды, сгруппировать данные |
| Создание таблиц | Создать новый Sheet с нужной структурой |
| Формулы | Написать SUM, SUMIF, VLOOKUP и другие формулы по описанию на русском языке |
| Добавление данных | Дописать строки, обновить значения |

### Что работает хуже

| Ограничение | Что делать |
|-------------|-----------|
| Графики и диаграммы | Данные подготовит Cowork, визуализацию добавьте в Sheets вручную |
| Apps Script и макросы | Отдельная задача вне MCP |
| Таблицы 10 000+ строк | Предварительно фильтруйте данные |
| Тонкое форматирование | Cowork записывает данные, финальное оформление — вручную |

Результат: новый лист в Google Sheets с рейтингом менеджеров, понедельной динамикой и формулами для итогов. Всё создано через MCP без ручной работы.
