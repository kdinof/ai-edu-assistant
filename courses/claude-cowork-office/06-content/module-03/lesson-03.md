## Видео-скрипт

### Вступление (30 сек)

Здравствуйте! В прошлом уроке мы анализировали существующие таблицы. Теперь обратная задача — создать новую таблицу с нуля. Бюджет проекта, трекер задач, табель рабочего времени — любую структурированную таблицу, которая нужна для работы.

Разберём, как ставить задачу Cowork, чтобы получить таблицу с правильной структурой и рабочими формулами.

### Основная часть

**Почему «сделай таблицу» — плохое задание**

Представьте: вы просите коллегу «сделай таблицу для бюджета». Какую таблицу он сделает? С какими колонками? Какими формулами? На какой период? Скорее всего, результат придётся переделывать.

То же самое с Cowork. «Создай таблицу для бюджета проекта» — слишком абстрактно. Вот что нужно указать:

1. **Структура:** какие колонки и строки
2. **Формулы:** какие вычисления нужны
3. **Данные:** заполнить примерами или оставить пустым
4. **Формат:** заголовки, итоговые строки, листы

**Демонстрация: бюджет проекта**

Задание Cowork:

«Создай в Google Sheets таблицу "Бюджет проект Альфа Q2". Структура:
- Колонки: Категория, Описание, План (сум), Факт (сум), Отклонение (сум), Отклонение (%)
- Категории: Зарплаты, Аренда, Маркетинг, IT-инфраструктура, Командировки, Представительские, Прочее
- Формулы: Отклонение = Факт - План. Отклонение % = (Факт - План) / План. Итоговая строка с суммами по колонкам План, Факт, Отклонение
- Колонку План заполни нулями, Факт оставь пустой»

Что создаёт Cowork:
1. Новый Google Sheet с заданным названием
2. Заголовки колонок
3. 7 строк категорий
4. Формулы в колонках «Отклонение» и «Отклонение %»
5. Итоговую строку с SUM-формулами
6. Нули в колонке «План», пустые ячейки в «Факт»

Теперь вы заполняете колонку «План» реальными цифрами, а по мере расходования — колонку «Факт». Отклонения рассчитываются автоматически.

**Формулы: описывайте словами, не синтаксисом**

Вам не нужно знать синтаксис Google Sheets формул. Описывайте задачу обычным языком:

| Что вам нужно | Как сказать Cowork |
|---------------|-------------------|
| Сумма по колонке | «Формула суммы для колонки D» |
| Среднее по отделу | «Среднее значение продаж для строк, где отдел = "Маркетинг"» |
| Процент от общего | «В каждой строке — доля от общей суммы в процентах» |
| Условное суммирование | «Сумма расходов, где категория "Маркетинг" и месяц "март"» |
| Поиск значения | «Найти цену товара по его коду из таблицы на листе "Прайс"» |

Cowork сам выберет подходящую функцию — SUM, AVERAGE, SUMIF, SUMIFS, VLOOKUP, INDEX/MATCH — и вставит формулу в нужную ячейку.

**Шаблоны типовых таблиц**

Вот задания для таблиц, которые нужны в большинстве офисов. Используйте как основу и адаптируйте под себя:

**Трекер задач:**
«Создай Google Sheet "Трекер задач отдела". Колонки: Задача, Ответственный, Дедлайн, Статус (выпадающий список: Новая/В работе/На проверке/Готово), Приоритет (Высокий/Средний/Низкий), Комментарий. Добавь лист "Сводка" с подсчётом задач по каждому статусу и по каждому ответственному».

**План мероприятий:**
«Создай Google Sheet "План мероприятий Q2". Колонки: Мероприятие, Дата, Место, Бюджет (план), Бюджет (факт), Ответственный, Статус. Итоговая строка с суммой бюджетов. Отдельный лист "По месяцам" — сумма бюджетов по каждому месяцу».

**Табель рабочего времени:**
«Создай Google Sheet "Табель март 2026". Строки — сотрудники (10 строк, имена оставь пустыми). Колонки — дни месяца (1-31) плюс итого часов. Формула итого: сумма часов за все дни. Внизу — итого по каждому дню».

**Связь с документами из модуля 2**

Таблицы и документы работают в связке. Cowork может использовать данные из Google Sheets для создания документов в Google Docs — и наоборот.

Пример: «Возьми данные из таблицы "Бюджет проект Альфа Q2" и создай в Google Docs краткую справку для руководства: общий бюджет, топ-3 категории расходов, отклонения от плана».

Это пайплайн — тема модуля 4. Но уже сейчас помните: каждая таблица, которую вы создаёте, может стать источником данных для документов и отчётов.

### Заключение (30 сек)

При создании таблиц ключевое — детальное описание: структура колонок, нужные формулы, начальные данные. Описывайте формулы словами — Cowork сам выберет правильную функцию.

В следующем уроке — практика: вы создадите и проанализируете таблицу на своих реальных рабочих данных.

## Текстовое сопровождение

### Что указывать при создании таблицы

Абстрактное «сделай таблицу» даёт непредсказуемый результат. Указывайте четыре элемента:

1. **Структура** — колонки, строки, листы
2. **Формулы** — какие вычисления нужны (описывайте словами)
3. **Данные** — заполнить примерами или оставить пустым
4. **Формат** — итоговые строки, заголовки, сортировка

### Формулы обычным языком

Вам не нужно знать синтаксис. Описывайте задачу — Cowork подберёт функцию:

| Задача | Функция, которую использует Cowork |
|--------|-----------------------------------|
| Сумма по колонке | SUM |
| Среднее по условию | AVERAGEIF |
| Условное суммирование | SUMIF / SUMIFS |
| Процент от общего | Деление на SUM с фиксацией ячейки ($) |
| Поиск по коду | VLOOKUP / INDEX+MATCH |

### Шаблоны типовых офисных таблиц

**Бюджет проекта:** категории расходов, план/факт, отклонения в сумме и процентах, итоговая строка.

**Трекер задач:** задача, ответственный, дедлайн, статус (выпадающий список), приоритет, сводка по статусам.

**Табель рабочего времени:** сотрудники × дни месяца, итого по строкам и столбцам.

**План мероприятий:** мероприятие, дата, бюджет план/факт, помесячная сводка.

### Связь с модулем 2

Каждая таблица в Google Sheets — потенциальный источник данных для документов из Google Docs. Cowork может брать данные из Sheets и формировать отчёты в Docs автоматически.

## Ключевые выводы

1. **Детальное описание структуры** — ключ к хорошей таблице. Укажите колонки, строки, формулы и формат. «Сделай таблицу для бюджета» — плохо. Описание с конкретными колонками и формулами — хорошо
2. **Описывайте формулы словами**, а не синтаксисом. «Сумма расходов, где категория "Маркетинг"» — Cowork сам выберет SUMIF и напишет правильную формулу
3. **Таблицы и документы работают в связке.** Данные из Google Sheets можно использовать для создания отчётов в Google Docs — это основа пайплайнов в модуле 4
