## Видео-скрипт

### Вступление (30 сек)

Здравствуйте! Это финальный практический урок курса. Вы соберёте всё, что создали за модули 1-4, в работающий AI-офис: skills, workflow, обновлённый CLAUDE.md и Time Audit с реальными цифрами экономии.

### Основная часть

**Задание 1: Создайте минимум 3 skills (5 мин)**

Посмотрите на задания, которые вы выполняли в модулях 2-4. Выберите 3 задачи, которые вы делаете регулярно, и создайте для них skills.

Для каждого skill:
1. Возьмите задание, которое дало хороший результат
2. Убедитесь, что в нём есть все 4 элемента: Что → Откуда → Куда → Формат
3. Создайте skill через Settings → Skills → Add Skill или попросите Cowork

Попросите Cowork помочь:
> «Создай 3 skills для моих регулярных задач: [перечислите]. Для каждой — skill с полной инструкцией.»

Проверьте: запустите каждый skill и убедитесь, что результат соответствует ожиданиям:
> «[Название skill]»

**Задание 2: Создайте один составной skill (2 мин)**

Выберите рабочий процесс из нескольких шагов и создайте для него составной skill:

Примеры:
- **Конец недели:** сводка задач → отчёт → отправка руководителю
- **Подготовка к встрече:** данные → повестка → шаблон протокола
- **Конец месяца:** данные → сводка → отчёт → презентация

Создайте через Cowork:
> «Создай skill "[название]" с инструкцией: [опишите все шаги цепочки]»

**Задание 3: Time Audit «после» (3 мин)**

Откройте Google Sheets «Time Audit» из модуля 1. Выполните те же 3-5 задач с Cowork и запишите время:

> «Открой Google Sheets "Time Audit". Добавь колонки "Время после" и "Экономия". Я буду диктовать результаты замеров.»

Или заполните вручную и попросите Cowork посчитать:
> «Открой Google Sheets "Time Audit". Заполни колонку "Экономия" — разница между "Время до" и "Время после". Добавь строку "Итого" с суммой экономии в неделю.»

**Финальный чеклист AI-офиса**

Ваш AI-офис готов, если выполнены все пункты:

- [ ] **Cowork настроен** — Claude Desktop установлен, Cowork доступен
- [ ] **Google Workspace CLI подключён** — Claude видит файлы в Google Drive
- [ ] **CLAUDE.md актуален** — содержит контекст должности, правила и формат документов
- [ ] **Skills созданы** — минимум 3 skills для регулярных задач (в Settings → Skills)
- [ ] **Составной skill** — минимум 1 skill с цепочкой из 2+ шагов
- [ ] **Time Audit заполнен** — замеры «до» и «после» для 3-5 задач

Попросите Cowork создать чеклист:
> «Создай Google Doc "Чеклист AI-офиса" в папке "AI-офис" со списком всех компонентов и их статусом.»

**Что дальше**

Ваш AI-офис — не статичная система. Он будет расти:

- **Новые skills.** Каждый раз, когда находите задачу с повторяющейся структурой — добавляйте skill
- **Обновление CLAUDE.md.** При смене должности, проекта или правил компании — обновляйте контекст
- **Scheduled tasks.** Настройте автоматический запуск для самых частых задач
- **Новые интеграции.** Google Calendar, Gmail — Cowork работает со всеми сервисами Google Workspace

### Заключение (30 сек)

Поздравляю — вы завершили курс! У вас теперь есть работающий AI-офис: Claude Cowork с Google Workspace CLI, CLAUDE.md с контекстом вашей должности, библиотека skills для повторяющихся задач, workflow для рабочих процессов и Time Audit с реальными цифрами экономии. Используйте этот офис каждый день — и он будет становиться только эффективнее.
