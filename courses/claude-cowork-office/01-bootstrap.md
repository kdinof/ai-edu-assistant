# Bootstrap: Claude Cowork для офисных работников

> Дата: 2026-03-06
> Автор идеи: Кирилл

## 1. Тема курса

**Название (рабочее):** Claude Cowork для офисных работников

**Описание в 1-2 предложениях:** Курс по использованию Claude AI (Claude Code / Cowork) для ежедневных офисных задач — создание презентаций, отчётов, работа с таблицами, почтой и подготовка к встречам. Фокус на автоматизации рутины и измеримой экономии времени.

## 2. Целевая аудитория

**Кто этот человек?** Офисные работники: бухгалтеры, менеджеры проектов, маркетологи, финансисты, HR-специалисты. Работают в компаниях в Узбекистане.

**Уровень подготовки:** Новички в AI-инструментах. Часть пробовали ChatGPT как чат-бота для общения, но не используют AI системно для рабочих задач.

**Что уже умеет/знает:** Хорошо владеют офисными инструментами (Google Docs, Sheets, Slides и т.д.). Уверенные пользователи ПК.

**Главная боль/мотивация:** Много времени уходит на рутинные задачи (отчёты, письма, подготовка презентаций). Хотят автоматизировать рутину, чтобы больше времени посвящать стратегическим задачам и работе, где нужна экспертиза и время для размышлений.

## 3. Learning Outcomes

После прохождения курса студент сможет:

1. Писать эффективные промпты для рабочих задач, получая качественный результат в виде отчётов, писем, таблиц и презентаций
2. Самостоятельно автоматизировать сбор отчётов, создание презентаций и обработку рутинных задач с помощью Claude Code
3. Создавать презентации и структурированные документы с помощью Claude
4. Подготавливаться к встречам быстрее — повестка, тезисы, протокол
5. Измерить и продемонстрировать экономию времени от внедрения AI в свой рабочий процесс

## 4. Финальный артефакт

**Что студент создаст к концу курса:**

1. **Персональная библиотека промптов и автоматизаций** — набор готовых промптов и подходов, адаптированных под конкретные рабочие задачи студента
2. **Time Audit (до/после)** — измерение сколько часов уходило на рутинные задачи до курса vs. после, с конкретным списком автоматизированных задач

## 5. Параметры курса

**Желаемая длительность:** Мини-уроки по 5-15 минут

**Формат:** Видео (скринкаст с демонстрацией экрана) + текстовый дубль каждого урока

**Количество модулей:** 5 (максимум 6)

**Уроков в модуле:** Не более 5

## 6. Инструменты и технологии

**Какие инструменты студент будет использовать:**
- Claude Code / Claude Cowork — основной AI-инструмент
- Google Workspace: Sheets, Docs
- Презентации в формате PPTX
- Работа с PDF-документами (презентации и документы)

**Нужна ли подписка/лицензия:** Да, обязательна платная подписка Claude. Без неё курс пройти невозможно.

**Есть ли эксперт/ментор для курса:** Да, будет приглашён внешний эксперт. Эксперт записывает скринкасты по подготовленной структуре курса.

**Особые требования или ограничения:** Язык курса — русский. Перевод на узбекский после завершения основной версии.
