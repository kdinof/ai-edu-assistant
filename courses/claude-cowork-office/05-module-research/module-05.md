## 1. Claude Desktop skills and projects setup workflow automation 2026

Claude Desktop in 2026 offers **skills**, **projects**, and **workflow automation** capabilities that enable users to customize their AI experience and automate repetitive tasks.

## Skills and Customization

Claude Desktop features a **Customize section** that groups skills, plugins, and connectors in one unified location[2]. This allows you to extend Claude's functionality with custom tools and integrations tailored to your workflow. Skills can be configured through the settings menu to personalize how Claude responds and operates[1].

## Projects for Organized Workspaces

**Projects** function as custom workspaces where you can organize reference documents and maintain context for specific tasks[1]. This feature helps you compartmentalize different work areas, keeping relevant information and instructions together for more focused interactions with Claude.

## Workflow Automation with Cowork

**Cowork** is Claude Desktop's agentic capability system that automates knowledge work beyond coding[2]. Key automation features include:

- **Scheduled tasks**: Create both recurring and on-demand tasks that execute automatically[2]
- **Browser integration**: Control browser actions directly from Claude Desktop, allowing Claude to handle work in your browser without requiring you to switch windows[2]
- **Workflow recording**: Teach Claude your workflow by recording steps, and it learns to repeat them for repetitive browser tasks[2]
- **Multi-tab coordination**: Claude can work across multiple browser tabs simultaneously[2]

Cowork is available on Claude Desktop for macOS, with Pro and Max plan access[2]. The system runs locally on your computer in an isolated virtual machine, enabling direct access to local files and MCP (Model Context Protocol) integrations[2].

## Integration with Claude Code

Claude Code integrates with Cowork to provide **multi-agent project handling**, where Claude can spin up specialized subagents to tackle different parts of complex problems simultaneously[3]. Each subagent operates with its own context window, maintaining quality regardless of how many agents run in parallel[3].

## 2. Claude Cowork recurring tasks scheduled automation examples 2026

**Claude Cowork enables scheduled recurring automations** that execute tasks automatically on a defined schedule without manual intervention each time[1][2].

## How Scheduled Automations Work

You set up a task once by describing what you want done, then choose the schedule for execution[2]. Claude Cowork will run the task automatically according to your specifications and deliver the finished output for review[2].

## Real-World Examples

**Automated Social Media Management**

A hotel on Lake Garda configured Claude Cowork to generate and publish LinkedIn posts automatically[1]. The setup: *"Every morning at 9:00 AM, generate a LinkedIn post in Italian about sustainable tourism, using updated information about Lake Garda and our seasonal offers. Use a professional but welcoming tone. Publish automatically."* Claude executes this daily by searching for relevant information, creating original brand-aligned content, and publishing to the designated channel[1].

**Recurring Business Reports**

For an e-commerce business, you can schedule Claude to automatically connect to multiple platforms (Shopify, Google Analytics, Meta Ads, Mailchimp), extract data, create visualizations, and generate a complete PDF report with analysis and recommendations on a recurring basis[1].

**Daily Calendar and Email Management**

A civil lawyer in Naples can schedule a morning automation asking: *"What do I have on my calendar today? Summarize the important emails received yesterday from clients. If there are urgent appointment requests, suggest three available slots this week."* This runs automatically each morning[1].

## Key Advantage

Unlike one-time automations, scheduled tasks eliminate repetitive setup work—you configure the automation once and Claude handles execution consistently according to your defined schedule[2].
