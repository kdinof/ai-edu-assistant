> Статус: Draft

## Паспорт курса

| Параметр | Значение |
|----------|----------|
| Название | AI Office Bundle |
| Slug | ai-office-bundle |
| Длительность | ~3 часа |
| Формат | Видео-скринкасты (5-15 мин/урок) + текстовый дубль |
| Уровень | Начинающий |
| Пререквизиты | Базовые офисные навыки (почта, документы, таблицы). Опыт с AI не требуется |
| Финальный артефакт | Персональная AI-система: 5+ настроенных Skills для своих рабочих задач, измеренная экономия времени на реальных задачах |

## Цели обучения (Learning Outcomes)

После прохождения курса студент сможет:

1. **Создавать** рабочие документы (отчёты, письма, протоколы) с помощью Claude в 2-3 раза быстрее, чем вручную
2. **Анализировать** табличные данные и длинные документы через Claude — извлекать ключевые показатели, находить тренды, формировать сводки
3. **Конструировать** повторяемые Claude Skills для своих регулярных задач — превращая разовые промпты в стабильные рабочие инструменты
4. **Оценивать** качество AI-output и корректировать промпты для получения точного результата, соответствующего рабочим стандартам

## Целевая аудитория

### Основной сегмент
- **Кто:** Офисные работники без технического образования — маркетологи, HR-специалисты, бухгалтеры, менеджеры проектов, ассистенты, администраторы
- **Боль:** Тратят часы на рутинную работу с документами и таблицами. Слышали об AI, но не знают как встроить его в свои конкретные задачи. 63% боятся использовать AI неправильно
- **Мотивация:** Освободить время от рутины, выглядеть продуктивнее на работе, не отстать от коллег, которые уже используют AI
- **Контекст:** Учатся в свободное время или по инициативе работодателя. Self-paced, короткие сессии по 15-30 минут

### Дополнительный сегмент
- **Кто:** Руководители малого бизнеса / фрилансеры, которые совмещают несколько ролей и хотят автоматизировать административную нагрузку

## Scope

### Включено
- Работа с текстовыми документами: создание, редактирование, переписывание (отчёты, письма, протоколы)
- Анализ данных: таблицы (CSV, Excel), длинные документы (PDF, DOCX) — извлечение, сводки, тренды
- Claude Skills: создание повторяемых workflow для регулярных задач
- Промпт-инжиниринг для офисных задач: структурированные запросы, итеративная доработка
- Безопасность данных: что можно/нельзя загружать в Claude, анонимизация

### НЕ включено
- **Программирование и Claude Code** — курс для нетехнических пользователей, CLI-инструменты за скоупом
- **MCP-интеграции** (Notion, Google Workspace, CRM) — требуют техническую настройку, усложняют курс для начинающих
- **Другие AI-инструменты** (ChatGPT, Gemini, Copilot) — глубина в одном инструменте вместо поверхностного обзора многих
- **Управление командой / enterprise deployment** — курс для индивидуального пользователя
- **Создание изображений и мультимедиа** — не является сильной стороной Claude

## Топики и модули (высокоуровнево)

| # | Модуль | Фокус | Время | Артефакт модуля |
|---|--------|-------|-------|-----------------|
| 1 | Первый результат за 5 минут | Быстрый win: первый рабочий документ с Claude. Интерфейс, базовый промпт, загрузка файла | 30 мин | Рабочее письмо/документ, созданный с Claude |
| 2 | Документы на автопилоте | Создание отчётов, протоколов, писем. Шаблоны промптов, итеративная доработка, tone of voice | 40 мин | 3 шаблона промптов для своих документов |
| 3 | Данные без формул | Анализ таблиц и длинных документов. Загрузка CSV/Excel/PDF, извлечение, сводки, тренды | 40 мин | Аналитическая сводка из своих рабочих данных |
| 4 | Skills: промпт → инструмент | Создание Claude Skills для повторяемых задач. Превращение лучших промптов в стабильные workflow | 35 мин | 2-3 рабочих Skill для своих задач |
| 5 | Моя AI-система | Сборка персональной системы: библиотека Skills, замер экономии времени, план расширения | 35 мин | Персональная AI-система + замер Time Saved |

**Сквозной проект:** Студент работает со своими реальными рабочими задачами с первого модуля. Каждый модуль расширяет набор инструментов: Модуль 1 (первый документ) → Модуль 2 (шаблоны документов) → Модуль 3 (данные) → Модуль 4 (Skills из лучших промптов) → Модуль 5 (система).

## Практика

Каждый модуль содержит задание **"сделай это сейчас со своими данными"**:

| Модуль | Практическое задание |
|--------|---------------------|
| 1 | Создай рабочий документ, который тебе нужен прямо сейчас (письмо, ответ, заметка) |
| 2 | Возьми документ, который ты пишешь регулярно, и создай шаблон промпта для него |
| 3 | Загрузи свою рабочую таблицу или отчёт и получи аналитическую сводку |
| 4 | Преврати свой лучший промпт из модулей 2-3 в Claude Skill |
| 5 | Замерь время на 3 реальных задачах: с Claude и без. Собери свою систему |

## Безопасность данных

Встроено в каждый модуль по контексту:

- **Модуль 1:** Что можно и нельзя загружать в Claude. Правило: никаких паролей, персональных данных клиентов, конфиденциальных финансовых данных
- **Модуль 2:** Как анонимизировать примеры в промптах (замена имён, цифр)
- **Модуль 3:** Правила работы с таблицами: удалять PII перед загрузкой, использовать агрегированные данные
- **Модуль 4:** Безопасность в Skills: не хранить чувствительные данные в инструкциях

| Конкурент | Платформа | Сильные стороны | Слабые стороны |

| AI for Everyone (DeepLearning.AI) | Coursera | Бесплатный, авторитетный автор (Andrew Ng) | Теоретический, инструмент-агностичный, нет практики |
| Google AI Essentials | Coursera | Бесплатный, сертификат Google | Фокус на Google-экосистему, поверхностный |
| AI for Productivity (Coursera) | Coursera | Покрывает несколько инструментов | Multi-tool = неглубоко по каждому, нет Skills |

| YouTube tutorials | YouTube | Бесплатные, актуальные | Разрозненные, нет структуры и прогрессии |

**Дифференциация OSNOVA:**
- **Один инструмент — глубоко:** Только Claude, но с полным покрытием офисных сценариев (vs. поверхностный обзор 5 инструментов)
- **Практика на своих данных:** Каждый модуль = задание с реальными рабочими файлами (vs. демо-сценарии)
- **Skills как финальный артефакт:** Студент уходит с набором рабочих инструментов, а не только знаниями (vs. "теперь вы знаете что такое AI")
- **Контекст Узбекистана:** Примеры и кейсы релевантные локальному рынку

## Риски и допущения

- **Риск:** Claude меняет интерфейс/функции → скринкасты устаревают → **Митигация:** Текстовый дубль легко обновить, скринкасты делать модульными (по 5-10 мин), не привязываться к pixel-perfect UI
- **Риск:** Студенты без подписки Claude Pro не смогут выполнять задания → **Митигация:** Курс доступен через AI Bundle от OSNOVA, включающий доступ к Claude
- **Риск:** Нетехнические пользователи испугаются "AI" в названии → **Митигация:** Первый модуль — quick win за 5 минут, снижает порог входа
- **Риск:** Skills — относительно новая фича Claude, может измениться → **Митигация:** Skills используются в модулях 4-5, к этому моменту студент уже освоил базу и может адаптироваться
- **Допущение:** Студенты имеют доступ к рабочим файлам (документы, таблицы) для практики
- **Допущение:** Claude Pro / AI Bundle доступен в Узбекистане без ограничений
