# Stage 08: Lesson Summaries

## Задача

Сгенерировать текстовые описания (рекапы) для каждого урока. Эти описания размещаются под видео и дают студенту сжатую выжимку пройденного материала.

## Вход

- `06-content/module-{nn}/lesson-{nn}.md` — контент уроков

## Выход

- `06-content/module-{nn}/lesson-{nn}_summary.md` — текстовое описание к каждому уроку

## Подход

Используй агента `lesson-summary-writer` через Agent tool:

```
Agent(
  subagent_type: "lesson-summary-writer",
  prompt: "Сгенерируй текстовые описания для всех уроков курса {course_slug}. Директория контента: courses/{course_slug}/06-content/"
)
```

Агент автоматически:
1. Сканирует все модули и уроки в `06-content/`
2. Запускает параллельные sub-agents (один на урок)
3. Генерирует структурированные описания по шаблону
4. Сохраняет файлы рядом с контентом уроков

## Формат выходного файла

```markdown
# [Название урока]

## О чём этот урок
[1-2 предложения — суть урока]

## Ключевые тезисы
- [Тезис 1]
- [Тезис 2]
- [Тезис 3]
- ...

## Краткое содержание
[2-4 абзаца — сжатый пересказ. Без воды, только суть.]
```

## Quality Gate

- Каждый ключевой топик из урока представлен в описании
- Нет выдуманного контента — всё из исходного файла
- Длина: 150-400 слов
- Нейтральный профессиональный тон
