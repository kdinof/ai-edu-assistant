# /pipeline — Content Pipeline

Запусти или возобнови Content Pipeline для создания курса.

## Инструкции

### Шаг 1: ВСЕГДА сначала спроси пользователя

**Перед любыми действиями** выполни:

1. Просканируй папку `courses/` — найди все подпапки с `pipeline-state.json`
2. Для каждого найденного курса прочитай `pipeline-state.json` и определи:
   - `course_slug`
   - `current_stage` и его статус
3. Сформируй список вариантов для AskUserQuestion:
   - Для каждого существующего курса: `"Продолжить: {slug} (этап: {current_stage})"`
   - Последний вариант всегда: `"Создать новый курс"`
4. Задай вопрос через **AskUserQuestion**:
   - question: "Какой курс запустить?"
   - options: сформированный список

**Не продолжай pipeline автоматически. Жди ответа пользователя.**

### Шаг 2: Действуй по выбору

**Если выбран существующий курс:**
1. Прочитай оркестратор: `.claude/skills/content-pipeline/SKILL.md`
2. Прочитай `courses/{slug}/pipeline-state.json`
3. Определи текущий стейдж и продолжи по инструкциям оркестратора (раздел "Возобновление")

**Если выбрано "Создать новый курс":**
1. Прочитай оркестратор: `.claude/skills/content-pipeline/SKILL.md`
2. Если пользователь предоставил идею курса в $ARGUMENTS — используй её
3. Если нет — спроси идею курса через AskUserQuestion (freeform)
4. Следуй инструкциям оркестратора (раздел "Новый курс")

Входные данные: $ARGUMENTS
