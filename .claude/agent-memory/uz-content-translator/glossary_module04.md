## Style decisions (Module 4)
- "Что → Откуда → Куда → Формат" consistently rendered as "Nima → Qayerdan → Qayerga → Format"
- Pipeline examples in code blocks kept in Uzbek (translated)
- Monetary amounts: "сум" → "so'm"
- Uzbek names used in examples: Alisher, Nilufar (kept from original)
